## Development

C compiler: MinGW-W64

[MinGW](https://winlibs.com/#download-release)

### Debug Build

Run `DEBUG BUILD`

### Release Build

Run `RELEASE BUILD`

### Build

1. Open this project

2. Run `DEBUG BUILD` or `RELEASE BUILD`

3. Your jar will appear in `./build`

- Cache Dir is `<userhome>/.yolbi`

## Custom

For custom build, please edit `YBuild.xml`
